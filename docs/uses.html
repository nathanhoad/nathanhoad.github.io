What I use to make games

January 2023

I've been asked a number of times about what I use to do things so here is a list that I can point people at.

This was last updated January 10th, 2023.

Development

  • Godot is my game engine of choice
  • Visual Studio Code for writing non-gdscript code
  • Dracula Theme for everything
  • GitHub for hosting my code
  • Notion for writing notes and todo lists

Art

  • Aseprite for pixel art
  • Inkscape for vector art
  • Krita for drawing on my Surface Pro 9

Sound/Music

  • Yeti X microphone
  • Reaper for music and sound
  • Spitfire BBC Symphony Orchestra Discover for big sounding music
  • Surge XT for synth instruments

Video

  • OBS Studio to capture game footage
  • DaVinci Resolve for editing videos

If you want to keep up to date with my adventures in game development then you should subscribe to my YouTube channel or join my Discord.
